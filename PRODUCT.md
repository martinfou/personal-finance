# Personal Finance Manager — PRODUCT.md

**Vision:** Simple, beautiful, private personal finance tracking for Canadians.

## Users
- Martin (primary), single user with multi-account potential
- French language interface
- Tech comfort: moderate

## Core Experience
- Dashboard shows monthly snapshot: income, expenses, balance
- Transactions are the atomic unit — add, categorize, review
- Budgets keep spending on track
- CSV import from Canadian banks reduces friction
- Savings goals motivate long-term progress

## Tone & Personality
- Professional but warm
- Financial but not intimidating
- Clear, direct, helpful
